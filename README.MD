# 2. Cài đặt
### 2.1 Cài đặt Backend:
- Cài đặt các thư viện cần thiết:
```bash
cd backend
npm install
```
- Chạy backend:
```bash
npm run start 
# hoặc 
npm run dev
```

### 2.2 Cài đặt Frontend:
- Cài đặt các thư viện cần thiết:
```bash
cd frontend
npm install
```
- Chạy frontend:
```bash
npm run dev
```

### 2.3 Cài đặt AI:
- Cài đặt các thư viện cần thiết:
```bash
cd ai-service
pip install -r requirements.txt
```
